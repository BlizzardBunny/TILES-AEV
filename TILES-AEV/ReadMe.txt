How to play the game:
1. Run the TILES-AEV.exe file
2. In the game, use WASD keys to move around.
3. Hold SHIFT to run.
4. Press ESC to exit the program.

~Source codes~
Player
-Player is the script that the Player sprite game object uses. It detects the key input related to movement.
CameraFollow
-CameraFollow is the script that the Main Camera game object uses. It manages the function that allows the camera to follow the player. It also handles the exit application feature.
